This figure shows the component-level result before the scalar score. Each value is relative to the scenario-level geometric mean. A value below 1.0 means lower hardware demand, and a value above 1.0 means higher demand. The 80 Hz case is strong mainly in Tdot, delta, and delta-dot, while Tw is closer to neutral. The 5 Hz case is weak across most components. So the score should be read as a summary of these component trade-offs.
