# Data included in this repository

This folder contains selected CSV outputs required to support the final analysis. Large intermediate files are intentionally not included.

Not included:

- optimizer attempt logs
- checkpoint folders
- per-step NPZ trajectory dumps
- raw large solver archives

Included:

- Main4 MPC hardware-search summary CSV files
- Main5 path-pool OCP refinement summary and permutation CSV files
- completion check summaries used for sensitivity analysis
